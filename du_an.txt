
         PhantomX Minecraft Launcher - PROJECT STRUCTURE


================================================================================
1. TỔNG QUAN DỰ ÁN
================================================================================

Tên dự án: PhantomX Minecraft Launcher
Mục tiêu: Launcher mã nguồn mở, hỗ trợ offline/online, mod manager, community hub
Ngôn ngữ: Python 3.12.x latest
Framework GUI: PyQt6 / CustomTkinter (chọn 1)
Backend API: FastAPI + PostgreSQL
Database local: SQLite + SQLAlchemy
License: GPL-3.0
Repository: GitHub


================================================================================
2. CÔNG NGHỆ SỬ DỤNG
================================================================================

CORE:
- Python 3.12.x latest
- PyQt6 (GUI) hoặc CustomTkinter + tkmacosx
- HTTPX (async HTTP client)
- aiofiles (async file I/O)
- SQLAlchemy 2.0 (ORM)
- Alembic (database migration)

NETWORK:
- aiohttp (WebSocket client)
- python-socks (proxy support)
- qbittorrent-api (P2P sharing)
- curl_cffi (anti-fingerprint)

AUTHENTICATION:
- msal (Microsoft authentication)
- python-jose (JWT)
- passlib (password hashing)
- python-keyring (secure credential storage)

MINECRAFT SPECIFIC:
- nbt (NBT file parsing)
- pyMcInfo (Minecraft protocol)

MODDING:
- fabric-installer (wrapper)
- forge-installer (wrapper)

GUI ENHANCEMENT:
- qdarkstyle (dark theme)
- qt-material (material design)
- pyqtgraph (performance graphs)
- qtawesome (icons)

UTILITIES:
- loguru (logging)
- sentry-sdk (error tracking)
- python-dotenv (env vars)
- psutil (system monitoring)
- pyinstaller (packaging)
- pytest (testing)

DEVOPS:
- docker-compose (API server)
- nginx (reverse proxy)
- redis (caching)
- prometheus (monitoring)
- grafana (visualization)
